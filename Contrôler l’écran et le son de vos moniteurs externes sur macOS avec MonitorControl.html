Contrôler l’écran et le son de vos moniteurs externes sur macOS avec MonitorControl

Posted on Sat, Nov 30, 2024 Tuto Apple Silicon macOS M1 M2 M3 M4

Introduction

Si vous utilisez un écran externe sur macOS, vous avez probablement rencontré la difficulté de régler la luminosité ou le volume via les réglages natifs. MonitorControl est une solution open-source qui permet de contrôler facilement ces paramètres pour tous vos écrans externes directement depuis la barre de menu.

Fonctionnalités

MonitorControl offre plusieurs fonctionnalités utiles :

• Luminosité : Ajustez la luminosité de votre écran externe comme vous le feriez sur un écran Apple.

• Volume : Réglez le volume de votre moniteur, une fonctionnalité souvent absente sur macOS pour les écrans non-Apple.

• Température des couleurs : Modifiez la température des couleurs de l’écran (si pris en charge).

L’application est compatible avec de nombreux modèles d’écrans externes, y compris ceux de marques autres qu’Apple, ce qui en fait une option idéale pour les utilisateurs disposant de plusieurs écrans.

Installation et configuration

L’installation de MonitorControl est simple :

1. Téléchargez l’application depuis GitHub.

2. Installez-la en faisant glisser l’application dans le dossier Applications.

3. Lancez l’application et accordez les autorisations nécessaires pour permettre à MonitorControl d’agir sur vos écrans externes.

4. Personnalisez votre expérience depuis la barre de menu en ajustant la luminosité, le volume et la température des couleurs.

Options avancées

MonitorControl permet également de lier les touches de réglage de luminosité de votre clavier à tous vos écrans externes et offre la possibilité de synchroniser la luminosité avec les moniteurs Apple, améliorant ainsi l’expérience d’utilisation pour les utilisateurs de configurations multiples.

Conclusion

Si vous travaillez régulièrement avec des écrans externes, MonitorControl est un must-have pour personnaliser vos réglages d’affichage et de son. Simple à installer et efficace, il transforme l’expérience d’utilisation de macOS avec des moniteurs non-Apple.
