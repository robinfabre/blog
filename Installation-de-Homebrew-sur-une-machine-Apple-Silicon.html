Installation de Homebrew sur une machine Apple M1

Posted on Wed, Mar 17, 2021 Apple Silicon M1 Homebrew Tuto

Homebrew est un gestionnaire de paquets pour Mac qui permet d’installer facilement des programmes via une ligne de commande du terminal.

Pour l’installer, je me suis rendu sur leur site et j’ai copié/collé la ligne de commande fourni dans le terminal :

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Il faisait partie des programmes ayant été rendu compatible avec les processeurs Apple Silicon et donc je ne pensais pas rencontrer de problème.

Il se trouve que lors du lancement de ma première commande j’ai eu le problème suivant :

zsh: command not found: brew

J’ai donc fait quelques recherches sur le net et voici la solution que j’ai trouvé :

  • Dans le terminal, se rendre au repertoire bin de Homebrew avec la commande suivante :
cd /opt/homebrew/bin/
  • Lancer la commande suivante :
export PATH=$PATH:/opt/homebrew/bin
  • Revenir à la racine avec la commande suivante :
cd ~/
  • Créer le fichier .zshrc avec la commande suivante :
touch .zshrc
  • Alimenter le fichier avec la commande suivante :
echo export PATH=$PATH:/opt/homebrew/bin >> .zshrc

À partir de là, Homebrew refonctionne et vous pouvez l’installation du programme que vous souhaitez.
